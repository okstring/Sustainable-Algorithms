https://www.acmicpc.net/problem/2609

```python
def gcd(a, b) -> int:
    if a % b == 0:
        return b
    else:
        return gcd(b, a % b)


def lcm(a, b) -> int:
    return int((a * b) / gcd(a, b))


input = list(map(int, input().split(" ")))
a, b = input[0], input[1]

print(gcd(a, b))
print(lcm(a, b))

```
