https://www.acmicpc.net/problem/2748

```python
N = int(input())
cache = [-1 for _ in range(0, N+1)]
def fibonacci(n):
  if cache[n] != -1:
    return cache[n]
  elif n == 0:
    cache[n] = 0
    return 0
  elif n ==1:
    cache[n] = 1
    return 1
  else:
    cache[n] = fibonacci(n - 1) + fibonacci(n - 2)
    return cache[n]

print(fibonacci(N))
```
